# Identidade Visual

## Direcao visual

O ConvertTextEasy usa uma identidade de produto tecnico com aparencia moderna, noturna e utilitaria.

Caracteristicas principais:

- visual de dashboard/editor
- paineis transluidos com blur
- contraste alto
- foco em leitura tecnica e interacao rapida
- tom mais funcional que institucional

## Paleta principal

Variaveis visuais encontradas em `style.css`:

- fundo base escuro: `#07111f`
- azul primario: `#60a5fa`
- azul forte: `#3b82f6`
- verde de acento: `#22c55e`
- texto claro: `#e5edf8`
- texto secundario: `#93a4bf`

Modo claro tambem existe e troca a base para cinzas e azuis muito suaves.

## Linguagem de superficie

O sistema visual trabalha com:

- superficies semitransparentes
- bordas suaves com baixa opacidade
- gradientes discretos
- sombras largas e macias
- bordas arredondadas generosas

Isso aproxima o produto de uma mistura entre editor online, painel SaaS e ferramenta tecnica premium.

## Tipografia

Fonte principal declarada:

- `Space Grotesk`
- fallback para `Inter` e sans-serif de sistema

Efeito visual esperado:

- geometria moderna
- boa leitura em UI
- personalidade mais tech/editorial do que corporativa tradicional

## Componentes visuais marcantes

- header com efeito glassmorphism
- cards com hover elevado
- editor com fundo proprio e barra de acoes compacta
- pills, badges e icones simples para reforcar utilidade
- rodape com gradiente proprio

## Tema claro e escuro

O projeto foi pensado com theming real por variaveis CSS.

Principio atual:

- dark mode como experiencia-base da marca
- light mode como opcao de acessibilidade e preferencia pessoal

## Percepcao de marca

A identidade transmite:

- rapidez
- praticidade
- confianca tecnica
- produto online moderno
- utilidade para SEO, texto e dev

## Diretrizes para futuras alteracoes

- manter o azul como eixo principal de navegacao e destaque
- usar o verde apenas como acento secundario
- evitar visuais excessivamente genericos de blog comum
- preservar o contraste alto nas areas de editor e card
- manter o padrao de paineis, cantos arredondados e blur como assinatura do produto
